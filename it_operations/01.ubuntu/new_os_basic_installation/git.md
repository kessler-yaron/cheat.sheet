# git install 

#### 1) os system update
  ```sh
sudo apt-get update -y ; apt-get upgrade -y
```

#### 2) install git
  ```sh
sudo apt-get install git -y
```

#### 3) install git-gui -y
  ```sh
sudo apt-get install git-gui -y
```

#### (*) All git Installation Steps
 ```sh

sudo apt-get update -y ; apt-get upgrade -y

## 2) install git
sudo apt-get install git -y

## 3) install git-gui
sudo apt-get install git-gui -y

```
